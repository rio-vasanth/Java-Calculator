# Java-Calculator

A simple Java console calculator that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- Java
- Scanner Class

## How to Run
1. Compile the program:
   ```bash
   javac Calculator.java
   ```
2. Run the program:
   ```bash
   java Calculator
   ```
